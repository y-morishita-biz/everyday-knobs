VALUE

いますぐ、気軽に。

インストール不要

ブラウザを開けばすぐ。アカウントもサーバーも、アプリのインストールも要りません。

思いのままの形

丸・多角形・波型・指針。ローレットも目盛りもドームも、組み合わせは無限大です。

実際に挿さる・刷れる

実寸公差と公差テストピースで、印刷したノブが本当に軸へ挿さる。机上の空論にしません。

SHOWCASE

このツールで、できること。

形・テクスチャ・天面・印刷・共有。パラメータの組み合わせでどこまで化けるかを、まずは眺めてみてください。

HOW IT WORKS

3ステップで、書き出しまで。

01選ぶ

作例ギャラリーから、つくりたい形に近いものをタップ。それが出発点になります。

02調整

スライダーとトグルで自分好みに。迷ったら 🎲 ランダムで思いがけない形に出会えます。

03書き出す

STL / STEP / PNG でダウンロード。印刷して、軸にそのまま挿せます。

GALLERY

作例ギャラリー

気になる形をタップして、そのままジェネレーターへ。

ENCODERS

お手持ちの部品で。

軸寸法はメーカー STEP から実測。公差はスライダーで微調整できます。

ALPS EC11

φ6 ・ Dカット軸

EC1110120005 に対応。平面（Dカット）の向きまで実寸でフィットします。

ALPS EC12E

φ6 ・ 中空軸

EC12E085 に対応。中空軸の内径もカバー。嵌合クリアランスは調整可。
