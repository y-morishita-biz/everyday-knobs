# まいにちのぶ 開発ログ（#まいにちのぶ）

「1日1ノブ」の活動ログ。1日 = 1機能実装 + 1作例ノブ + 1 X（旧Twitter）投稿（**#まいにちのぶ**）。
新しい機能を実装し終えたら、このファイルの先頭に新しい Day エントリを **追記**する（運用手順は
[`CLAUDE.md`](../CLAUDE.md) を参照）。

エントリの雛形：

```markdown
## Day NN — YYYY-MM-DD — <ひとことタイトル>

- **実装した機能**：何を足したか（ジェネレーター側の機能）
- **作例ノブ**：その機能で作った1つのノブ（対象軸・寸法・狙い）
- **検証**：どう動作確認したか（build / 実機 / スクショ）
- **Xポスト下書き**：`#まいにちのぶ Day NN …`（投稿そのまま）
- **ブランチ / コミット**：`feat/day-NN-<slug>` / 主要コミットhash
```

## Day 1 — 2026-06-10 — MVP ジェネレーターが動いた

- **実装した機能**：ブラウザで動くパラメトリック・ノブ・ジェネレーターの MVP。
  - 本体（円柱）＋軸穴（ネガ形状）をブーリアンで生成
  - 軸タイプをトグルで切替：**EC1110120005（φ6 Dカット）** ⇄ **EC12E085（φ6 中空＝丸穴）**
  - スライダー：本体径 / 本体高さ / 軸穴深さ / 嵌合クリアランス → リアルタイムにプレビュー更新
  - 破綻形状（軸穴 ≥ 本体径、肉厚ゼロ等）をスライダー可動域（最小肉厚 1.5mm 確保）で防止
  - **STL / STEP エクスポート**
  - CAD カーネル（Replicad = OpenCASCADE WASM）を **WebWorker** で実行し UI を固めない。
    プレビューは Three.js。Vite + React + TypeScript。
- **作例ノブ**：円柱ノブ **φ20 × H16mm**、EC11 φ6 Dカット軸用、嵌合クリアランス 0.15mm。
  （ジェネレーターのデフォルト＝最初の1個）
- **検証**：
  - `npm run build`（型チェック＋バンドル）通過
  - ヘッドレス CAD で両軸タイプの boolean cut / STL / STEP 生成成功（STEP は `ISO-10303-21` 妥当）
  - 実機ブラウザ（Chromium）で目視確認：底面に **Dカットの軸穴**を確認、トグル・スライダーの
    リアルタイム更新、STL/STEP のダウンロード、コンソールエラー 0 件
- **Xポスト下書き**：
  > #まいにちのぶ Day1 🎛️
  > ブラウザで動くノブ・ジェネレーターのMVPができた。ALPS EC11(φ6 Dカット)とEC12E(中空)の
  > 軸穴をトグルで切替、本体径/高さ/クリアランスをスライダーで即プレビュー、STL/STEPで書き出し。
  > Replicad(WASM)+Three.js、サーバー不要のクライアントオンリー。
- **ブランチ / コミット**：（基盤期は `claude/bold-mccarthy-j97rug`） / `b3a5eb5`
