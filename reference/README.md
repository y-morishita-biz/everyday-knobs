# reference — 対象エンコーダの一次資料

このプロジェクトが対応するロータリーエンコーダの寸法・形状を、**メーカー公式 CAD（STEP）から
実測したリファレンス**として保管しています。生成ツール側のコードはここの値を「正」として
軸穴（ネガ形状）の寸法を決めます。

> **メーカー配布 CAD（STEP/PDF）と利用条件ファイルは、再配布の懸念を避けるため本リポジトリには
> 同梱していません。** 原データは各機種の ALPS 公式ページから入手してください（下表リンク）。
> 寸法は本ファイル末尾の「軸の確定寸法（STEP実測）」表と、識別用の外観プレビューのみを残しています。

## 対象エンコーダ（3機種）

要件定義 v1 では「EC11 のφ6軸3種（セレーション / 平軸 / 溝軸）」を対象としていましたが、
**仕様変更**により、以下の実在する機種を直接の対象とします
（EC12E2440301 は**対応準備中**＝寸法未確定）。

| 機種 | シリーズ | 軸タイプ | メーカーページ（CAD入手元） | 本リポジトリ |
|---|---|---|---|---|
| **EC1110120005** | EC11（金属軸 / メタルシャフト） | φ6 ソリッド軸 | [tech.alpsalpine.com](https://tech.alpsalpine.com/j/products/detail/EC1110120005/) | 実測寸法表・プレビュー |
| **EC12E085** | EC12E（中空 / インサート軸） | 中空シャフト | [tech.alpsalpine.com](https://tech.alpsalpine.com/j/products/category/encorders/sub/02/series/ec12e/) | 実測寸法表・プレビュー |
| **EC12E2440301** | EC12E（絶縁軸 / 低背） | 樹脂・絶縁軸（取付やや複雑） | [tech.alpsalpine.com](https://tech.alpsalpine.com/j/products/detail/EC12E2440301/) | **寸法未確定**（STEP実測待ち） |

2機種は軸インターフェースが大きく異なります（片方はソリッド軸、もう片方は中空軸）。
これは「軸モジュールを差し替え可能に作る」という要件定義の設計方針を、最初から実物2種で
検証する良いケースになります。

## データに関する注意

- **STEP の単位はインチ**：`EC1110120005.step` は SolidWorks 由来（`automotive_design` /
  AP214）で、座標がインチで格納されています。寸法を読むときは **×25.4 で mm 換算**します。
  ざっと確認した範囲では軸の半径3.0mm（=**φ6**）の円筒面が含まれており、φ6軸であることは
  STEP から確認できます。
- **PDF は3Dビューのみ**：両PDFとも寸法線つきの2D図面ではなく、3DモデルをPDFに埋め込んだ
  ものです（Adobe Reader 等で回転表示可能）。正確な寸法は **STEP からの読み取り＋実物の
  ノギス実測**で確定します（要件定義の方針どおり）。
- **EC12E085 も STEP はインチ単位**：EC12E085 の STEP も同じく SolidWorks 由来（AP214）で
  座標はインチ。外径φ6.05・内径φ3.0 の**中空シャフト**（下表参照）。
- **ライセンス / 取り扱い**：寸法の元になったアルプスアルパイン社の配布 CAD（STEP/PDF）は、
  再配布の懸念を避けるため**本リポジトリには含めていません**。原データは公式ページから各自で
  入手してください。本リポジトリに残しているのは、自分で解析した**実測寸法表**と識別用の
  **外観プレビュー**のみです。

## 軸の確定寸法（STEP実測）

STEP の頂点・円筒面・平面配置を解析して読み取った値（**インチ→mm は ×25.4**）。
ノブ側の軸穴はこの値を基準に設計する。

### EC1110120005（EC11 金属軸）

| 項目 | 実測値 | 根拠／備考 |
|---|---|---|
| 軸径 | **φ6.0** | 円筒面 r=3.0 |
| 軸断面 | **Dカット（片面フラット）** | フラット面（PLANE）が軸心から **1.5mm** offset → **二面幅 4.5mm**。ALPS標準φ6平軸に一致 |
| フラット範囲 | 先端側 約8mm（Z≈12〜20） | 根元側 約7mm（Z≈5〜12）は全周φ6 |
| 先端 | 面取りあり（先端で半径が縮小） | |
| 軸の突出長 | 約15mm | 取付面〜先端（STEP Z≈5〜20） |
| 根元ボス／本体 | φ7 段付き、本体ボス φ10〜φ11、本体 約13mm角 | 干渉確認用 |
| 全高 | 約23.5mm | STEP Z範囲 −3.5〜20 |

### EC12E085（EC12E 中空／インサート軸）

| 項目 | 実測値 | 根拠／備考 |
|---|---|---|
| 軸形式 | **中空（インサート）シャフト** | |
| 外径 | **φ6.05** | 円筒面 r=3.025（ノブが被さる外周） |
| 内径（ボア） | **φ3.0** | 円筒面 r=1.5 |
| 軸断面 | キー／フラット付きプロファイル | 軸周りに複数のフラット面を検出（offset 約2.0〜2.95mm 等） |
| 本体ボス | φ10（φ7.6 段付き） | |
| 軸の突出長 | 約6.5mm | ボス上面〜軸先端 |
| 全高 | 約12mm | STEP Z範囲 −3.5〜8.45 |

> 解析メモ：両 STEP とも巨大なトリム／構築用座標を含むため、生の CARTESIAN_POINT 全体では
> なく **VERTEX_POINT（実頂点）＋ CYLINDRICAL_SURFACE / PLANE の配置**から寸法を取った。

### EC12E2440301（EC12E 絶縁軸・低背 ／ **寸法未確定・暫定**）

> ⚠️ **未実測**。下表は対応準備（Day32）時点の**暫定プレースホルダ**で、コード側
> （`app/src/cad/params.ts` の `SHAFTS.EC12E2440301`）にも同じ暫定値が `provisional: true`
> 付きで入っています。**メーカー公式 STEP を実測して確定値に差し替える**こと。

| 項目 | 暫定値 | メモ／要確認 |
|---|---|---|
| 軸形式 | **絶縁軸（樹脂）／低背** | EC12Eの insulated shaft。金属Dカットではない |
| 軸断面（ソケット） | **セレーション（仮）** `teeth=20 / 歯丈0.5` | スプライン/十字/平 のどれかを STEP で確定。コードは `socket.kind` で切替可 |
| 軸径 | φ6.0（仮） | 要実測 |
| 軸の突出長 | 約9mm（仮） | 低背なので短め想定。要実測 |
| 推奨軸穴深さ | 7mm（仮） | `recommendedHoleDepth` |
| 本体ボス径/高さ | 未測 | ノブ裾の逃げ設計に使用予定（`bossDiameter`/`bossHeight`） |

> 取付が「ややこしい」理由：絶縁軸はノブ側に**キー/スプライン噛み合わせ**を要することが多く、
> 単純な丸穴/Dカットでは嵌合しない可能性が高い。確定後に `socket.kind` を適切な断面
> （`serrated` / `double-flat` / 必要なら新規 `cross`）へ設定する。
