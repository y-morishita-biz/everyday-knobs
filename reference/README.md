# reference — 対象エンコーダの一次資料

このプロジェクトが対応するロータリーエンコーダの寸法・形状を、**メーカー公式 CAD（STEP）から
実測したリファレンス**として保管しています。生成ツール側のコードはここの値を「正」として
軸穴（ネガ形状）の寸法を決めます。

> **メーカー配布 CAD（STEP/PDF）と利用条件ファイルは、再配布の懸念を避けるため本リポジトリには
> 同梱していません。** 原データは各機種の ALPS 公式ページから入手してください（下表リンク）。
> 寸法は本ファイル末尾の「軸の確定寸法（STEP実測）」表と、識別用の外観プレビューのみを残しています。

## 対象エンコーダ（3機種）

軸インターフェースの異なる実在の3機種を直接の対象とします。ジェネレーターの軸モデルは
`socket.kind`（ローレット/Dカット/丸）で各断面を生成します（`app/src/cad/params.ts`）。

| 表示名（コードID） | シリーズ／タイプ | 軸断面（socket） | データ元 |
|---|---|---|---|
| **EC11シリーズ互換**（`EC11COMPAT`） | EC11互換・φ6 ストレート・ローレット軸 | `serrated`（φ6 ローレット） | EC11系標準・モデル化 |
| **EC12E24404A8**（`EC12E24404A8`） | EC12E・12型 絶縁軸・**Dカット**（背高） | `dcut`（φ6・片面フラット） | EC12E-200.STEP（mm） |
| **EC12E1240301**（`EC12E1240301`） | EC12E・絶縁**中空**軸・**丸**（低背） | `round`（φ6中空） | EC12E-085.step（インチ）＋外形図 |

> 命名整理（Day32）：旧 `EC11`/`EC12E`/スプライン版 `EC12E2440301` を廃し、上記3機種へ再編。
> 軸の保存キー（注文コード/URL）も新IDに統一（公開告知前のため互換は気にしない）。

## データに関する注意

- **STEP の単位は混在**：`EC12E-085.step`（→EC12E1240301）は SolidWorks 由来・AP214 で
  **インチ**格納（**×25.4 で mm 換算**）。`EC12E-200.STEP`（→EC12E24404A8）は **mm** 格納。
  単位は各STEPの `CONVERSION_BASED_UNIT('INCH')` / `SI_UNIT(.MILLI.,.METRE.)` で判定。
- **ファイル名が紛らわしい**：ALPS の EC12E シリーズ配布 CAD は名前が共通の体裁になりがちで、
  ファイル名と機種が一致しないことがある。**中身の形状で機種を判別**する（提供者確認済み）。
- **PDF は3Dビューのみ**：寸法線つき2D図面ではなく3Dモデル埋め込み。正確な寸法は
  **STEP読み取り＋外形図＋実物ノギス**で確定（要件定義の方針どおり）。
- **ライセンス / 取り扱い**：元になったアルプスアルパイン社の配布 CAD（STEP/PDF）は、
  再配布の懸念を避けるため**本リポジトリには含めていません**。原データは公式ページから各自で
  入手してください。本リポジトリに残しているのは、自分で解析した**実測寸法表**のみです。

## 軸の確定寸法（STEP実測）

STEP の頂点・円筒面・平面配置を解析して読み取った値（**インチ→mm は ×25.4**）。
ノブ側の軸穴はこの値を基準に設計する。

> 解析メモ：STEP は巨大なトリム／構築用座標を含むため、生の CARTESIAN_POINT 全体ではなく
> **VERTEX_POINT（実頂点）＋ CYLINDRICAL_SURFACE の配置**から寸法を取った。EC12E-200 は mm 単位、
> EC12E-085 はインチ単位（×25.4）。ファイル名は EC12E シリーズ共通の体裁で紛らわしいが、
> **中身の形状で機種を判別**している（提供者確認済み）。

### EC11シリーズ互換（`EC11COMPAT`／φ6 ストレート・ローレット軸）

| 項目 | 値 | 備考 |
|---|---|---|
| 軸径 | **φ6.0** | EC11系標準 |
| 軸断面（ソケット） | **ローレット（serrated）** `teeth:20 / 歯丈0.5` | ストレートの刻み軸にノブを圧入。`socket: serrated` |
| 軸の突出長 | 約15mm | EC11系標準想定 |
| 推奨軸穴深さ | 12mm | `recommendedHoleDepth` |

> ※歯数/歯丈は実物に合わせ調整可（専用STEP未取得のためモデル値）。圧入の効きは公差テストピースで詰める。

### EC12E24404A8（`EC12E24404A8`／12型 絶縁軸・Dカット・背高 ／ EC12E-200.STEP・**mm**）

| 項目 | 実測値 | 根拠／備考 |
|---|---|---|
| 軸形式 | **絶縁軸（樹脂）・Dカット** | 片面フラット |
| 軸径 | **φ6.0** | 円筒面 φ6.0/φ6.06 |
| 軸断面（ソケット） | **Dカット**：フラット ≈軸心1.5mm（二面幅4.5・φ6標準） | `socket: dcut, flatDistance:1.5`。**二面幅は実物/図面で要確認** |
| 軸の突出長 | 約17mm（背高） | 実頂点 Z≈0〜17.7 |
| 全高 | 約23mm | 実頂点 Z 範囲 −5.75〜17.7 |

### EC12E1240301（`EC12E1240301`／絶縁中空軸・丸・低背 ／ EC12E-085.step・**インチ**＋外形図）

| 項目 | 実測値 | 根拠／備考 |
|---|---|---|
| 軸形式 | **絶縁中空軸・丸（低背）** | 外形図：丸軸（スプラインではない） |
| 外径 | **φ6.0**（ブッシング φ6.6±0.1） | 図面 ø6±0.1／STEP 円筒 φ6.04 |
| 内径（ボア） | **φ3.1±0.05** | 図面 ø3.1／STEP φ3.0 |
| 軸断面（ソケット） | **C字キャビティ＋中央キー＋平頭ポスト**（`socket: hollow`） | 提供ノブSTEPの輪郭を実測トレース（下記） |
| 軸の突出長 | 約6.2mm（低背） | 実頂点 Z≈2.3〜8.45 |
| 本体ボス | φ10（φ7.6 段付き）・高さ≈2mm | `bossDiameter`/`bossHeight` |
| 全高 | 約12mm | 実頂点 Z 範囲 −3.5〜8.45 |

> 訂正メモ：当初この EC12E-085 STEP を「12歯スプライン」と誤読したが、**外形図で丸軸と確定**。
> セレーション断面（serrated）は EC11シリーズ互換のローレット軸の方に割り当てた。
>
> **取付形状（提供ノブSTEP `low_knob_a` の天面Z=3.45 輪郭を実測トレース）**：丸穴ではなく
> **C字キャビティ**。以下を `socket.kind = "hollow"` で再現（`postFlatY` / `key` で形状指定）：
> - **外周キャップ φ6.14** を **y=2.25 で平らにカット**（上が開いた C 字）
> - 開口に**中央キー（舌）**が上から下りる：**幅2.8（x±1.4）／ y=1.5 まで**（向き決め＝回り止め）
> - **中心ポスト φ3.1**、**上面を y=0.55 で平ら**にカット（D 断面）
> - 取付深さ ≈2.55mm（Z 0.90〜3.45）→ コードでは `fixedHoleDepth: 2.55` で固定
>
> 生成 void と参考頂点が一致することを確認済み。`knob_EC12E085_for_dotmatrix40_low` はノブ＋
> エンコーダの嵌合アセンブリで、相互の収まり確認に使用。
